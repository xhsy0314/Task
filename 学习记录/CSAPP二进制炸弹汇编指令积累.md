```
cmpl   $0x7,0x8(%rsp)		# 第一个数必须小于7，否则爆炸
ja     400fad				# 引爆炸弹
```
以上两句汇编等同于 if (*rsp+8) > 7, 跳转到0x400fad， 因此第一个数必须不大于7。

```
jmpq *0x402470(,%rax,8)
```

是一个间接跳转指令, 可以看出这段代码是典型的switch语句，跳转表就存在于0x402470。%rax取值为[0, 7]，代表switch语句中8条不同的case。
